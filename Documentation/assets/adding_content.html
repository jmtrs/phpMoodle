If you are working with a Moodle version prior to 2.7, there will be certain issues where the text editor won't react the way you expect (for example deleting the font awesome icons, see also "Styling your site" > "Font awesome").

So if your are working with Moodle version 2.5 or 2.6, I recommend installing the Atto editor. Atto is a javascript text editor built specifically for Moodle and it is the default text editor in Moodle from 2.7 onwards.
