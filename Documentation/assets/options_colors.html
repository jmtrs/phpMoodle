Theme Options

Configurable Options

To access the configurable options for Theme Lambda go to Administration > Site Administration > Appearance > Themes > Theme-Lambda.

Color Settings

For each element you can either use the color picker or enter the color code.

Note:
When specifying colours, you can even use advanced colors with transparency values such as rgba(208,33,33,0.5).

Setting up the colors in Moodle:
